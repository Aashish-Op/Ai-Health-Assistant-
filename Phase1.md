- `8480-6` → (VitalType.systolic_bp, "mmHg")
- `8462-4` → (VitalType.diastolic_bp, "mmHg")

`FHIRParser` class:

```python
class FHIRParser:
    def __init__(self, bundle: dict) -> None:
        """Index all bundle entries by resourceType for O(1) lookup."""
        self._resources: dict[str, list[dict]] = {}
        self._logger = get_logger(__name__)
        for entry in bundle.get("entry", []):
            resource = entry.get("resource", {})
            rtype = resource.get("resourceType")
            if rtype:
                self._resources.setdefault(rtype, []).append(resource)

    @classmethod
    def from_file(cls, path: Path) -> FHIRParser: ...

    @classmethod
    def from_bytes(cls, data: bytes) -> FHIRParser:
        """Parse from raw bytes. Raises FHIRParseError on invalid JSON."""

    def extract(self) -> PatientContext:
        """Main extraction entry point. Calls all private extractors.
        Logs total parse duration at DEBUG level.
        Raises FHIRParseError if Patient resource is absent."""

    def _extract_patient(self) -> dict: ...        # returns raw Patient resource
    def _extract_demographics(self, patient: dict) -> dict: ...
    def _extract_conditions(self) -> list[Condition]: ...
    def _extract_medications(self) -> list[Medication]: ...
    def _extract_allergies(self) -> list[Allergy]: ...
    def _extract_observations(self) -> tuple[list[LabResult], list[Vital]]: ...
    def _parse_bp_panel(self, obs: dict, recorded_at: datetime) -> list[Vital]: ...
    def _get_loinc_code(self, resource: dict) -> str | None: ...
    def _get_coding_by_system(self, codings: list[dict], system_substr: str) -> str | None: ...
    def _parse_date(self, raw: str | None) -> date | None: ...
    def _parse_datetime(self, raw: str | None) -> datetime | None: ...
    def _map_lab_flag(self, interpretation_code: str | None) -> LabFlag | None: ...
```

Rules:
- Condition filtering: include only where `clinicalStatus.coding[0].code in {"active", "recurrence", "relapse"}`
- Medication filtering: include only where `status == "active"`
- Observation splitting: LOINC in VITAL_LOINC_CODES → Vital; numeric valueQuantity → LabResult
- Lab sort: `recorded_at` descending, cap at 20
- Vital sort: `recorded_at` descending, cap at 15
- Every field extraction wrapped in defensive try/except — a missing field logs at DEBUG and returns None/default, never raises
- Only raise `FHIRParseError` if the bundle has no Patient resource at all

## CODE QUALITY STANDARDS — NON-NEGOTIABLE

**Async:** Every function that does I/O must be `async`. No `time.sleep()` — use `asyncio.sleep()`. Never call sync I/O from an async function without `asyncio.to_thread()`.

**Error handling:** Never let an unhandled exception reach the client. Register FastAPI exception handlers for each domain exception in `main.py`. Every handler returns a structured `ErrorResponse` with `error_code`, `message`, and the current `request_id`.

**Logging:** Every service method logs entry at DEBUG and exit at DEBUG with timing. Business events log at INFO. Unexpected errors log at ERROR with `exc_info=True`. Never log raw patient data — log `patient_id` only.

**Type hints:** 100% coverage. `from __future__ import annotations` at top of every file. Use `X | None` not `Optional[X]`. Never use `Any` without a comment.

**Pydantic:** Use `model_validator`, `field_validator`, `computed_field` where appropriate. Models are immutable after creation (`model_config = ConfigDict(frozen=True)` on data models).
